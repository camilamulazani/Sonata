# SONATA
Este jogo foi desenvolvido em C# para Unity como projeto de situação de aprendizado durante o curso técnico em programação de jogos digitais do SENAI-SC
---
## Visão Geral:
Um RPG de turno onde o jogador controla uma jovem banjoísta que explora um mundo musical, enfrentando outros músicos em combates épicos para coletar partituras e levar a arte de volta para a sua casa.
